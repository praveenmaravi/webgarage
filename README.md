## 🚀 Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/your-username/webgarage.git
cd webgarage
````

### 2. Install Dependencies

```bash
npm install
# or
yarn install
```

### 3. Setup Environment

Create a `.env` file:

```env
NEXT_PUBLIC_OPENAI_API_KEY=your_openai_key
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
```

### 4. Run the Dev Server

```bash
npm run dev
```
